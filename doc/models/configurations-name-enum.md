
# Configurations Name Enum

## Enumeration

`ConfigurationsNameEnum`

## Fields

| Name | Description |
|  --- | --- |
| `ACCOUNT` | The specified configuration will be used for bank accounts integrations. |
| `BATCHTXN` | The specified configuration will be used for processing transaction batches. |
| `BINREPORT` | The specified configuration will be used for bin reports integrations. |
| `BOARD` | The specified configuration will be used for boarding merchants. |
| `CHARGEBACK` | The specified configuration will be used for everything related to chargebacks. |
| `DECISION` | The specified configuration will be used for everything related to decisions. |
| `DECISIONBOARD` | This holds configuration for running boarding decision policy on entity through Risk MS. |
| `DECISIONTXN` | DecisionTxn is used for running transaction decision policy through the Risk MS. |
| `ECHECKTXN` | The specified configuration will be used to get reports of echeck transactions as well as getting additional information about those transactions. |
| `ENCRYPTION` | Holds the configuration for running Google and Apple Pay encryption/decryption through the Payrix Encryption Service. |
| `OMNITOKEN` | Holds the configuration parameters for omnitoken services. |
| `ONLINETXN` | The specified configuration will be used to process transactions online without a batch. |
| `ONLINETXNSOCKET` | OnlineTxnSocket. |
| `PAYMENTUPDATES` | PaymentUpdates. |
| `PAYOUT` | The specified configuration will be used for everything related to payouts. |
| `PAYOUTCREDIT` | Holds the configuration for processing payout credit through the integration service. |
| `PAYOUTDEBIT` | Holds the configuration for processing payouts debit through the integration service. |
| `PLAID` | Holds the configuration parameters for Plaid services (connecting accounts, getting identity, etc.). |
| `REPORTS` | The specified configuration will be used to import processed transactions. |
| `REGISTER` | Register. |
| `SAFERPAYMENT` | Holds the configuration parameters for safer payment VAS. |
| `UNREGISTER` | Unregister. |
| `VARSHEETRETRIEVAL` | Holds the configuration parameters for varsheet retrieval. |
| `WATCHLIST` | The specified configuration will be used to receive a watch list by the OFAC. |
